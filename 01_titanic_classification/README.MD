# Titanic Survival Prediction

## Problem
Celem projektu jest przewidzenie, czy pasażer Titanica przeżył katastrofę, na podstawie danych historycznych.

## Dane
Dataset (Kaggle): Titanic: Machine Learning from Disaster

## Model
AutoML – PyCaret (classification)

## Wynik
Najlepszy model: Logistic Regression  
Accuracy: 0.8209

## Wnioski
Model potrafi z dużą skutecznością przewidywać przeżycie pasażera.
Najlepszy model został zapisany do pliku w celu dalszego wykorzystania.
