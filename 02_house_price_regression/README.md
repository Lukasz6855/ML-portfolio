# House Price Prediction

## Problem
Celem projektu jest przewidywanie cen domów na podstawie ich cech.

## Dane
Dataset (Kaggle): House Prices - Advanced Regression Techniques 

## Model
AutoML – PyCaret (regresja)

## Wynik
Najlepszy model: Gradient Boosting Regressor  
Metryka błędu: MAE 17276.3461; RMSE 28313.9245

## Wnioski
Model pozwala oszacować cenę domu z rozsądnym błędem predykcji.
